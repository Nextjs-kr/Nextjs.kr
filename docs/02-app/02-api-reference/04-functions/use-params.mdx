`useParams`는 경로의 [동적 매개변수](/docs/app/building-your-application/routing/dynamic-routes)를 읽을 수 있게 해주는 **클라이언트 컴포넌트** 훅입니다. 어떤 경로의 동적 매개변수는 현재 URL 값으로 채워집니다.

```tsx filename="app/example-client-component.tsx" switcher
'use client'

import { useParams } from 'next/navigation'

export default function ExampleClientComponent() {
  const params = useParams()

  // Route -> /shop/[tag]/[item]
  // URL -> /shop/shoes/nike-air-max-97
  // `params` -> { tag: 'shoes', item: 'nike-air-max-97' }
  console.log(params)

  return <></>
}
```

```jsx filename="app/example-client-component.js" switcher
'use client'

import { useParams } from 'next/navigation'

export default function ExampleClientComponent() {
  const params = useParams()

  // Route -> /shop/[tag]/[item]
  // URL -> /shop/shoes/nike-air-max-97
  // `params` -> { tag: 'shoes', item: 'nike-air-max-97' }
  console.log(params)

  return <></>
}
```

## 매개변수

```tsx
const params = useParams()
```

`useParams`는 어떤 매개변수도 받지 않습니다.

## 반환값

`useParams`는 현재 라우트의 [동적 매개변수](/docs/app/building-your-application/routing/dynamic-routes)를 포함하는 객체를 반환합니다.

- 객체의 각 프로퍼티는 활성화된 동적 세그먼트입니다. (동적 세그먼트에 맞추어 각각의 프로퍼티가 자동으로 생성됨)
- 프로퍼티의 이름은 세그먼트의 이름이고, 프로퍼티의 값은 세그먼트에 채워진 값입니다.
- 프로퍼티의 값은 [동적 세그먼트의 타입](/docs/app/building-your-application/routing/dynamic-routes)에 따라 문자열 또는 문자열 배열이 될 수 있습니다.
- 라우트가 동적 매개변수를 포함하고 있지 않다면, `useParams` 는 빈 객체를 반환합니다.
- `pages`에서 사용하는 경우, `useParams`는 `null`을 반환합니다.

예시

| Route                           | URL         | `useParams()`             |
| ------------------------------- | ----------- | ------------------------- |
| `app/shop/page.js`              | `/shop`     | `null`                    |
| `app/shop/[slug]/page.js`       | `/shop/1`   | `{ slug: '1' }`           |
| `app/shop/[tag]/[item]/page.js` | `/shop/1/2` | `{ tag: '1', item: '2' }` |
| `app/shop/[...slug]/page.js`    | `/shop/1/2` | `{ slug: ['1', '2'] }`    |
